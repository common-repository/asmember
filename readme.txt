asmember is a membership plugin and adds membership functionality to your site. Only members can view protected content.

== Description ==

=asMember ist ein einfach zu nutzendes Membership-Plugin für Wordpress. Es erweitert die Wordpress Benutzerwaltung und 
stellt über Mitgliedschaften die Möglichkeit zur Verfügung, Inhalte nur für bestimmte Nutzergruppen zugänglich zu machen.

- eigene Formulare für Registrierung, Login und Passwort-Reset
- Mitgliederbereich mit einem eigenen Menü
- Mitgliedschaften, entweder frei oder kostenpflichtig
- Buchungsmodul für Mitgliedschaften
- Newsletter
- Erweiterungsmodul für Downloads (Zusatzprodukt kostenpflichtig)


= asMember is an simply and easy-to-use Plugin to create a member area for your WordPress site =

With asMember you can protect your posts, that only members of your site can view the restricted content.
Non-members viewing a protected page will be prompted to login.

= Shortcodes for login and register =

You can integrate login and register form in your site with shortcodes, e.g. in pages, posts or sidebars.

= Shortcodes for membership-area =

Your members can edit their own data with a accoount-form. To use this form, you set a shortcode to a page.

= Members-List and Member-Profile =

Members can view a memberlist and an member-profile-page of every member. To integrate this pages use shortocdes.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/asmember` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the asMember->Options screen to configure the plugin
4. Create Pages vor Login, Register, Members, Member, Dashboard and Account, with the Shortcodes
